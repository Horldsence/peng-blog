# GitHub OAuth 授权说明

## 为什么要使用 GitHub 账号登录？

为了提供更好的评论体验，本博客支持使用 GitHub 账号发表评论。通过 GitHub 登录，您可以：

- ✅ 无需注册额外账号
- ✅ 自动显示您的 GitHub 头像
- ✅ 使用您熟悉的 GitHub 用户名
- ✅ 安全可靠的 OAuth 2.0 授权

## 我们会获取哪些信息？

授权后，我们仅会获取以下基本信息用于评论显示：

- **用户名**：显示在评论旁边
- **头像 URL**：展示您的 GitHub 头像
- **邮箱地址**：用于 Gravatar 头像（可选）

**我们不会获取**：
- ❌ 您的私人仓库信息
- ❌ 您的代码和提交记录
- ❌ 任何其他敏感权限

## 授权流程

1. 点击 "使用 GitHub 登录" 按钮
2. 跳转到 GitHub 授权页面
3. 确认授权（仅需一次）
4. 自动返回本博客，开始发表评论

## 安全性说明

- 采用标准 OAuth 2.0 协议
- 不会保存您的 GitHub 密码
- 仅获取必要的公开信息
- 您可以随时在 GitHub 设置中撤销授权

## 数据使用

您的 GitHub 信息仅用于：
- 在评论中显示您的身份
- 展示您的头像
- 区分不同评论者

我们不会向第三方分享您的信息。
